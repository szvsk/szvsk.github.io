Minimal Debian Installation Guide

February 12, 2019

This guide was copy-pasted from ru-lang part of web, and placed here just like note for myself. Original source can be found here.

Install system from Expert Mode.

Required:

  • Make root password,
  • Accept use of non-free software,
  • Install system utilites only.

After installation there will be black screen with login-password request.

Install sudo

# login as super user
su
# install system administration program
apt install sudo
# add new user to sudo group
adduser username sudo
# reboot
reboot

Install packages

# Update list of packages
sudo apt update
# Install i3 windows manager
sudo apt install i3 menu
# Install graphical interface
sudo apt install xorg mesa-utils mesa-vulkan-drivers
# Install AMD/ATI firmware
sudo apt install firmware-amd-graphics
# Install NVIDIA firmware
sudo apt install nvidia-driver
# Install sound utilities
sudo apt install alsa-utils
# Initialize sound
sudo alsactl init
# Install WiFi utilities
sudo apt install wireless-tools wpasupplicant iw
# Install Midnight Commander
sudo apt install mc
# Install archive managers
sudo apt install p7zip-full unzip unrar-free
# Install web browser dillo
sudo apt install dillo
# Install feh image viewer
sudo apt install feh
# Install pdf viewer
sudo apt install zathura
# Install torrent client
sudo apt install transmission
# Install movie/audio player
sudo apt install mpv
# Install graphic informer
sudo apt install neofetch
# Install network utilities
sudo apt install net-tools hping3 iftop nmap vnstat ifstat
# Install system utilities
sudo apt install sysstat iotop atop powertop smartmontools
# Install compiler tools
sudo apt install build-essential
# Install file recovery tool
sudo apt install scalpel
# Install screenshot tool
sudo apt install scrot
# Start graphic mode
startx

Automatisation of startx

$ nano ~/.profile
# Add this line to the end of file
[[ -z $DISPLAY && $XDG_VTNR -eq 1 ]] && exec startx

Install Luakit (website).

Web browser with vim-like keybindings.

# First of all, install all dependencies
$ sudo apt install git
$ sudo apt install build-essential
$ sudo apt install gtk+-3.0
$ sudo apt install lua5.1
$ sudo apt install lua-filesystem
$ sudo apt install webkit2gtk-4.0
$ sudo apt install sqlite3
$ sudo apt install libluajit-5.1-dev
$ sudo apt install libsqlite3-dev
# Download latest version of luakit
$ git clone git://github.com/luakit/luakit
# Then we need go inside downloaded directory and compile program
$ cd luakit
$ make
# Then install program
$ sudo make install
# In case you need to uninstall luakit, it possible with next command
$ sudo make uninstall
